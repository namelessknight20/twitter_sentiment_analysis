# Twitter Sentiment Analysis

This project performs sentiment analysis on Twitter data and classifies tweets as:
- Positive
- Negative
- Neutral

## Tech Stack
- Python
- Scikit-learn
- NLTK / TF-IDF
- Jupyter Notebook

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the notebook

## Example
"I love this app" → Positive  
"this app is boring" → Neutral
